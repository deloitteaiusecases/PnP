## .1. Policy Statements

The below statements of policy are defined as the foundation of s view on open data and should be a guide on all actions in maintaining the standards of Open Data across . These statements are:
 shall create an Open Data Plan to identify and coordinate the publishing of its Open Datasets.
 shall launch awareness programs to make sure that the potential users are aware of the existence, nature and quality of Open Data
 The awareness programs should contain proper usage of data. It should also contain various positive social and economic benefits.
 The process to Identify Open datasets should be done in-line with Data Classification policy.
 An appropriate open data identification process needs to be designed and followed.
 The open data identification process should include:
o Identification and documentation of all data categorized as ‘public data’ as per Data Classification
o A data valuation process should be created and followed in order to have a uniform decision-making process to decide whether the particular dataset should be published as open data or not
o Unauthorized disclosure of personal or sensitive information which could create any security risk should be analyzed in line with Data Classification Policy.
 Standardized formats need to be created and followed for all the Open Data documents.
 The Open Data should be maintained in an appropriate manner
o The changes in the Open Data should be documented and the metadata updated periodically
o To ensure that the Open Data sets comply with the regulatory requirements, constant review of the Open Data sets need to be conducted.
o Key Performance Indicators need to be established in order to measure the progress of the Open Data Plan and to gather statistics.
o For easy access and navigation of the Open Data, the necessary metadata of Open datasets should also be maintained.
 An Open Data register is required to be maintained to log all the activities related to the Open Data.
