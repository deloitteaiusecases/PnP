## Cloud Computing and Hosting Cybersecurity

1. Requirements: Define, document, approve, and implement cybersecurity requirements related to the use of hosting and cloud computing services.
2. Data Classification: Classify data before hosting on cloud or hosting services.
3. Data Return: Ensure cloud service providers provide a separate virtual environment and return data in a usable format upon service completion.
4. Confidentiality Agreements: Ensure confidentiality and non-disclosure agreements are signed and monitored periodically.
5. Service Level Agreements: Sign service level agreements as per business requirements and monitor them periodically.
6. Security Policies: Ensure service providers follow Arabian Mills security policies and procedures.
7. Location: Ensure Arabian Mills information hosting and storage is inside the Kingdom of Saudi Arabia.
8. Periodic Review: Review cybersecurity requirements related to the use of hosting and cloud computing services periodically.
