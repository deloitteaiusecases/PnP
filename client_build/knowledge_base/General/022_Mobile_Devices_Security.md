## Mobile Devices Security

1. Requirements: Define, document, approve, and implement cybersecurity requirements for mobile devices security and bring your own device (BYOD).
2. Guidelines: Adopt guidelines and supporting security measures to manage risks introduced by using mobile devices.
3. Business Information: Ensure business information is not compromised when using mobile devices.
4. Unprotected Environments: Consider risks of working with mobile devices in unprotected environments in the mobile device guidelines.
5. Unauthorized Access: Implement protection to avoid unauthorized access to or disclosure of information stored and processed by mobile devices.
6. Teleworking: Develop and implement guidelines and supporting security measures to protect information accessed, processed, or stored at teleworking sites.
7. Data Separation: Develop and implement separation and encryption of Arabian Mills' data and information stored on mobile devices and BYODs.
8. Controlled Use: Include controlled and restricted use of mobile devices based on job requirements in mobile device security.
9. Secure Wiping: Develop secure wiping procedures for Arabian Mills' data stored on mobile devices and BYODs in cases of device loss, theft, or termination/separation.
10. Awareness: Provide continuous security awareness for mobile device users.
11. Periodic Review: Review cybersecurity requirements for mobile devices security and BYOD periodically.
